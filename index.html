Identificar os coeficientes da equação


Observe a equação do segundo grau no formato geral: ax² + bx + c = 0

Identifique os valores dos coeficientes a, b e c. Esses coeficientes determinam os termos da equação.


No caso da equação solicitada:


  • ax² + bx + c = 0

Calcular o discriminante (Δ)


O discriminante (Δ) é calculado pela fórmula: Δ = b²- 4.a.c.

Substitua os valores dos coeficientes a, b e c na fórmula e calcule o valor do discriminante.


  • Δ = b² - 4.a.c
  • Δ = b²- 4.a.c
  • Δ = b - 4.a.c
  • Δ = b - 4.a.c
  • Δ = resultado

Raízes

Analisar o valor do discriminante


Dependendo do valor do discriminante, a equação pode ter diferentes tipos de solução:


  • Se Δ > 0, a equação tem duas raízes reais e distintas.
  • Se Δ = 0, a equação tem duas raízes reais e iguais.
  • Se Δ < 0, a equação não tem raízes reais, apenas raízes complexas.

Calcular as raízes da equação


Se o valor do discriminante for maior que zero (Δ > 0), você pode calcular as duas raízes usando as fórmulas:


  • x1 = (-b + √Δ) / (2.a)
  • x2 = (-b - √Δ) / (2.a)

Se o valor do discriminante for igual a zero (Δ = 0), você pode calcular a única raiz usando a fórmula:


  • x = -b / (2.a)

Se o valor do discriminante for menor que zero (Δ < 0), a equação não possui raízes reais.
